# Developer Quickstart: Dashboard Update

This feature involves updating the `Dashboard.tsx` (or related components) to adopt the Editorial Vault design syntax.

1. Ensure the dev server is running with `npm run dev` and navigate to the dashboard page.
2. Review the Google Stitch "Wallet Management" file for UI layout parameters.
3. Validate typography overrides (Manrope headers, Inter body) and UI spacing constraints.
4. Verify elevation using `offset-y: 8px` shadow parameters over hard borders.
